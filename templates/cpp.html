Data Structures Final Project

This was a final project for a data structures class. We were instructed to compare add and search times of USPS tracking numbers using various data structures we would implement without using libraries. During this project I chose to go beyond the requirements of the project by implementing a low maintenance trie structure capable of constant time lookups and adds. github link

Image Processing Optimization

This was part of a project to optimize code that would apply filters to a bmp image. I was able to achieve a speedup of about 25-30x on average by making use of techniques like loop unrolling, thread parallism, compact data structures, and more efficient algorithms. The image to the left shows an example original image and the application of 4 different filters. In the repository which I have included a link to you can find a FilterMain.cpp file which contains the bulk of the code used to apply filters to images.

Shell Development

Using the standard C/C++ library I implemented a simple shell program (tsh.c), capable of operating in a similar manner to common linux shells. The shell makes use of common techniques like forks, execs, and signal handlers to run jobs. You can follow the included github link to a repository containing my shell program. Clone the repository and then run make to compile the shell program which can be run as ./tsh. github link
